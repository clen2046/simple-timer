# 资源文件说明

## 必要文件

### 1. 默认提醒音乐
- 文件名：`default_alarm.mp3`
- 格式：MP3、WAV或OGG格式
- 用途：当没有选择自定义音乐时使用的默认提醒音

### 2. 应用图标
- 文件名：`icon.ico`
- 格式：ICO格式（Windows图标）
- 用途：应用窗口图标和系统托盘图标

## 如何获取

### 对于默认提醒音乐：
1. 可以使用任何简短的音频文件（建议1-5秒）
2. 可以录制或下载简短的提醒音
3. 格式支持：MP3、WAV、OGG等（pygame.mixer支持格式）

### 对于应用图标：
1. 可以使用在线工具将PNG转换为ICO格式
2. 建议尺寸：16x16、32x32、64x64像素
3. 可以使用简单的钟表或计时器图标

## 测试使用
在开发测试阶段，程序会在文件不存在时使用系统蜂鸣声作为后备。
